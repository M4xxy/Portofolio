Human Stress Detection using NLP

Project Information

This data science project focuses on detecting human stress levels from text using Natural Language Processing (NLP) and machine learning. The process involved cleaning and preprocessing text data, converting text into numerical features using TF-IDF vectorization, and training multiple classification models to find the best performer. The final model accurately predicts stress levels based on written text.

Technologies Used

  • Python
  • Kaggle Notebooks
  • Scikit-learn (Machine Learning)
  • NLTK (NLP Toolkit)
  • Pandas & NumPy
  • Matplotlib & Seaborn
