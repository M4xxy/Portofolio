WatchGoods - A Watch Marketplace

Project Information

WatchGoods is a HTML/CSS comcept for a modern e-commerce platform specializing in luxury and vintage watches. The goal was to create a clean, responsive, and intuitive user interface using HTML/CSS, This was my first HTML/CSS project. This project focuses on simple product presentation, user navigation, and a seamless shopping experience.
