Suitmedia - Front-End Technical Assessment

Project Information

A multi-screen mobile application built as a technical skill assessment for Suitmedia. This project demonstrates a complete user flow, including a login screen with palindrome-checking logic, a welcome screen, and a third screen that fetches and displays a paginated list of users from a live API. The app was built with Next.js and TypeScript, showcasing modern web development practices.

Technologies Used

  • Next.js (React Framework)
  • TypeScript
  • Tailwind CSS
  • REST API Integration
